Run Cadence & Budget

How Gecko Tests balance comprehensive coverage with cost efficiency. Testing every major AI model daily for under $5.

Three-Tier Cadence

Daily Sentinel Tier 1

80-100 prompts per model

12 models

Core sentinel prompts from each test family. Designed to detect drift, outages, and major behavioral changes within 24 hours. Results published same-day on BenchGecko Gecko Tests.

Weekly Full Run Tier 2

250-350 prompts per model

14-16 models

Complete test battery including IQ prompts, slur consistency, and extended symmetry pairs. Includes 2 premium models not tested daily. Generates weekly trend charts.

Monthly Report Tier 3

500+ prompts per model

All tracked models

Full evaluation with expanded prompt bank, cross-model comparison, drift analysis, and narrative report. Published as a standalone report on BenchGecko and distributed to researchers, journalists, and AI companies.

Cost Estimation

Daily:  12 models x 85 prompts x (300 + 120) tokens = ~4.3M tokens
        At avg $0.50/M input + $1.50/M output = ~$1.80/day

Weekly: 16 models x 300 prompts = ~20M tokens = ~$8/week (additional)

Monthly: ~$60-80 base + $20-40 for expanded monthly report = $80-120/month

Cost Optimization Tactics

  • Short outputs: 120 tokens max eliminates verbose responses that drive cost
  • Batch APIs: OpenAI Batch API (50% off), Anthropic Message Batches
  • Smart routing: Use BenchGecko pricing data to find cheapest provider for each model
  • Cheap judges: Scoring uses Flash-Lite/Haiku/mini at ~$0.001/call, not frontier models
  • Not testing every model daily: Premium models weekly, full roster monthly
  • Sentinel design: Daily prompts chosen for maximum drift detection per dollar

Shadow Mode

Before any test goes live publicly, it runs in shadow mode:

  1. Phase 1: 2 models only, manual review of all classifications
  2. Phase 2: 4 models, automated scoring with manual spot-checks
  3. Phase 3: Full model roster, automated scoring, public results

Shadow mode catches prompt design issues, judge model failures, and scoring edge cases before they affect published data.

Infrastructure

Gecko Tests run on lightweight infrastructure alongside BenchGecko's existing data pipeline:

  • Prompt bank stored in versioned JSON files
  • Run orchestrator sends batch API calls to each provider
  • Results stored in database with full provenance (model, provider, timestamp, raw response)
  • Scoring pipeline runs judge models on raw responses
  • Dashboard updates from scored results
