Gecko Tests Methodology

Technical documentation for BenchGecko's daily AI behavior testing framework. Version 0.4

What are Gecko Tests?

Gecko Tests send identical prompts to every major AI model and measure behavioral patterns that existing benchmarks ignore: censorship, bias, political drift, moral reasoning, and real-life helpfulness. Built by BenchGecko, the data layer of the AI economy.

Why Behavior Testing Matters

Most AI benchmarks measure what models can do. MMLU tests knowledge. HumanEval tests coding. MATH tests arithmetic. These are capability benchmarks. You can compare thousands of them on BenchGecko's model rankings.

But nobody systematically tests what models will do when humans ask difficult questions. A model that scores 90% on MMLU but refuses to factually discuss a historical event has a behavioral problem that no capability benchmark captures.

Gecko Tests fill this gap. Same prompts, same models, raw answers, public data.

The 7 Core Tests

Censorship Index Daily

Which models refuse legitimate questions? 7 subscores measuring refusal rate, over-refusal, safety accuracy, unsafe compliance, moralizing, directness, and asymmetry.

Race Bias Index Daily

Do models apply different standards when only the race or ethnicity variable changes? Mirror-pair testing across hiring, policing, housing, education, and healthcare scenarios.

Test family details

AI Political Compass Daily

Where does each model sit on economic and social axes? Policy questions scored on a 2D compass. Not ideology detection -- orientation measurement.

Test family details

AI IQ Test Weekly

Fluid reasoning, pattern recognition, spatial logic. Not knowledge retrieval. Prompt bank rotated weekly to prevent contamination.

Test family details

Would AI Let People Die? Daily

Trolley problems, triage scenarios, rule-vs-survival conflicts. Scored on engagement depth, not moral position.

Test family details

Slur Double Standard Test Weekly

Do models treat slurs consistently across groups? Same question structure, different target groups. Measuring consistency, not permissiveness.

Test family details

Model Drift Index Computed

Derived from all other tests. Tracks when a model's censorship threshold shifts, when its political orientation moves, or when a provider silently updates behavior. Week-over-week and month-over-month deltas. Monitored alongside model updates on BenchGecko's model tracker.

Design Principles

  • Neutral tone. We publish data, not editorials. "Here are the answers" not "this model is evil."
  • Expected behavior labels. Every prompt has a pre-defined expected behavior. A model refusing dangerous content gets credit. A model refusing factual history gets flagged. See the expected behavior system.
  • Fairness protocol. If we test hard questions about one country, we test hard questions about every relevant country. If we test one ideology, we test the opposite. Five formal rules. See the fairness protocol.
  • Reproducibility. Temperature 0 or 0.2. Short outputs (120 tokens max). Fixed prompt bank. Same parameters across all providers.
  • Transparency. Methodology published. Prompt bank versioned. Raw answers archived. Everything except active IQ prompts is public.
