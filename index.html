Where the Data Comes From

BenchGecko aggregates data from authoritative sources and standardizes it into a single queryable dataset.

Official Provider APIs

Pricing, rate limits, and model metadata pulled directly from OpenAI, Anthropic, Google, Meta, Mistral, and other provider documentation.

Published Benchmark Results

Scores from peer-reviewed papers, official model cards, and standardized evaluation suites. Each score links back to its source.

Open Evaluation Harnesses

Results from LM Evaluation Harness, HELM, Open LLM Leaderboard, and other reproducible evaluation frameworks.

Community Leaderboards

Data from Chatbot Arena, LiveBench, and other community-driven evaluations that capture real-world model performance.

Agent Evaluation Suites

SWE-bench, WebArena, GAIA, and other agent-specific benchmarks that test autonomous task completion.

MCP Ecosystem

Model Context Protocol server registries, GitHub repositories, and community catalogs for tool and skill tracking.

Build with the BenchGecko API

Programmatic access to the full dataset. Query models, scores, pricing, and agents with a simple REST API. Use it to power your own AI tooling, dashboards, and decision engines.

  • Full model metadata: parameters, context window, release date, provider
  • Benchmark scores with source attribution
  • Real-time pricing across all commercial providers
  • Agent capabilities and evaluation scores
  • Comparison endpoints for side-by-side analysis
  • JSON responses, no authentication required for public data
API Documentation → 
# Get model details and scores
curl https://benchgecko.ai/api/v1/models/gpt-4o

# Compare two models
curl https://benchgecko.ai/api/v1/compare?a=claude-4&b=gpt-4o

# List all models with pricing
curl https://benchgecko.ai/api/v1/models?include=pricing

# Search agents by capability
curl https://benchgecko.ai/api/v1/agents?q=coding

# Get benchmark leaderboard
curl https://benchgecko.ai/api/v1/benchmarks/mmlu/scores

Engineering Teams

Find the best model for your specific task. Filter by benchmark category (coding, reasoning, math, multilingual), compare pricing, and evaluate the tradeoff between performance and cost before committing to an API provider.

Product Managers
